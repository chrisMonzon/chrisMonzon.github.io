PROJECTS

> ColorTaiko!

A web-based game I'm contributing to under the Illinois Mathematics Lab (IML).

Working with Professor Igor Mineyev and 5 other students, we're building an interface to help produce counterxamples to the Kaplansky conjectures.

Stack: React, JavaScript

> JolliBuy

An online market promoting and empowering Filipino small businesses. As a part of PSA (Philippine Student Assoication)'s Tech Team, I led the frontend development of this project.

Stack: React, Python, JavaScript, Django

> Custom Rubik's Cube Simulator

A graphics simulation program allowing users to upload .obj files to be converted into a fully interactable & playable, 2x2x2 Rubik's cube

Stack: Javascript, WebGL

> Building Block Simulator

A graphics simulation program allowing users to create and manipulate cubes in a 3D space

Stack: C++, OpenGL

> Excursia

A website built by me and three other students that web scrapes and displays prices of specific flights from Kayak based on user input.

Stack: Flask, Python
